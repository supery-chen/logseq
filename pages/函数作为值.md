- Go语言可以很灵活的创建函数，并作为值使用。以下实例中我们在定义函数中初始化一个变量，该函数仅仅是为了使用内置函数math.sqrt()
- ```go
  package main
  
  import (
     "fmt"
     "math"
  )
  
  func main(){
     /* 声明函数变量 */
     getSquareRoot := func(x float64) float64 {
        return math.Sqrt(x)
     }
  
     /* 使用函数 */
     fmt.Println(getSquareRoot(9))
  
  }
  ```
